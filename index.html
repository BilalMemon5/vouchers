    How to publish the customer tracking page (free, one-time setup)

    1. Create a free account at github.com if you don't have one.
    2. Create a new repository named e.g. vouchers, and turn on GitHub Pages in its Settings → Pages (branch: main, folder: /root).
    3. Click "Download Tracking Page" above for each customer — it saves one HTML file named after their voucher number.
    4. Upload that file into your vouchers repository (drag and drop on the GitHub website works fine).
    5. Your customer's page is now live at https://yourname.github.io/vouchers/UB-101818.html — paste that base address into the "Your hosting web address" box on the left, and the QR code updates automatically.
    Tip: you only do steps 1–2 once. After that, for every new customer you just fill the form, click Download Tracking Page, and drag the file into the same GitHub repository.
